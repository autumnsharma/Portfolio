# Design System: Radnaabazar Developer Portfolio

## Visual Identity
A high-contrast, "hacker-aesthetic" developer portfolio featuring a dark theme with vibrant neon accents and technical typography.

## Color Palette

### Core Colors
- **Background (Deep Space):** `#0A0B0E` (Deep charcoal/black with subtle grid pattern)
- **Primary Accent (Neon Mint):** `#00FFA3` (Used for headings, active states, and glowing borders)
- **Secondary Accent (Soft Teal):** `#14B8A6` (Used for secondary buttons and accents)
- **Surface:** `#111827` (Used for cards and elevated sections)

### Text Colors
- **Heading 1:** `#00FFA3` (Neon Mint)
- **Heading 2/Body Lead:** `#FFFFFF` (Pure White)
- **Body Text:** `#9CA3AF` (Muted Gray)
- **Muted/Caption:** `#6B7280` (Dark Gray)

## Typography

### Font Stack
- **Monospace (Primary):** `Fira Code`, `JetBrains Mono`, `ui-monospace`, `SFMono-Regular`, `Menlo`, `Monaco`, `Consolas`, `"Liberation Mono"`, `"Courier New"`, `monospace`
- **Sans-Serif (Secondary):** `Inter`, `ui-sans-serif`, `system-ui`

### Type Scale
- **Display 1 (Main Hero):** 4.5rem / 72px (Bold)
- **Heading 1:** 3rem / 48px (Bold)
- **Heading 2:** 2rem / 32px (Semi-Bold)
- **Body Large:** 1.25rem / 20px
- **Body Standard:** 1rem / 16px
- **Caption:** 0.875rem / 14px

## Components & Patterns

### 1. Hero Section
- **Grid Background:** A subtle `+` or dot grid pattern overlaying the deep background.
- **Title Block:** Large monospace greeting ("Hello I'm") followed by the name in Neon Mint.
- **Profile Image:** Circular avatar with a rotating, dashed neon border animation.

### 2. Buttons
- **Primary (Outline):** Neon Mint border (`1px`), uppercase monospace text, subtle glow on hover.
- **Secondary:** Muted background with high-contrast text.

### 3. Navigation
- **Header:** Clean horizontal layout with uppercase monospace links.
- **Active State:** Neon Mint underline with matching text color.
- **Language Switcher:** Pill-shaped button with neon mint background.

### 4. Stats Bar
- **Layout:** Bottom-aligned horizontal flex container.
- **Styling:** Large white numbers with smaller, muted descriptive text.

## Motion & Interactivity
- **Profile Border:** Continuous clockwise rotation animation on the dashed border.
- **Hover Effects:** Subtle scale-up and increased glow intensity for social icons and CTA buttons.
- **Transitions:** Smooth color and opacity transitions (approx. 200ms) for interactive elements.

## Design Principles
- **Clarity & Precision:** Use of monospace fonts to emphasize technical expertise.
- **High Contrast:** Neon accents on deep dark backgrounds for maximum visual impact.
- **Negative Space:** Generous padding around sections to focus attention on the core messaging.
